Introduction

This is a simple HTML document to demonstrate basic CSS styling.

This assignment introduces me to the foundational principles of CSS, how to style web content, apply essential styling properties, and understand the powerful concept of the CSS Box Model. I'll practice writing clean, organized CSS that brings structure and visual appeal to an HTML page.

Objectives

My goal is to create a visually styled web page using only CSS. I will apply basic styling rules to text, backgrounds, and layout elements, and demonstrate an understanding of how the CSS Box Model affects spacing and sizing on the page.

Conclusion

By completing this assignment, I will gain a solid understanding of how to use CSS to enhance the presentation of HTML content, making it more engaging and user-friendly.
